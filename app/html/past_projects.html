Ultimate Pi | September 16th, 2016 | Github Frontend & Github Backend | Deployed Here

Description

Full-Stack responsive web app that enables the client to use a configured Raspberry Pi as an IR remote

Project Employs:
  • Angular.js
  • Secure authentication system with registration
  • Custom built angular components
  • Sleek and intuitive remote UI
  • Webpack bundling
  • SCSS styling
  • Bootstrap
  • Font-awesome for Icons
  • TDD: Karma w/jasmine for frontend testing
  • TDD: Mocha with chai for backend testing
  • Deployed on Heroku

Fake Sports Real Money | August 11th, 2016 | Github Repo | Deployed API

Description

Pure node.js app for fantasy sports league commissioners which after initial setup will send a daily text and email reminder to members with overdue balances that they need to pay

Project Employs:
  • Twilio API for daily text reminders
  • Node-mailer for sending the daily email reminders
  • Cron for time tracking and to send daily reminders on overdue users
  • Express middleware
  • TDD
  • Deployed on Heroku

Nosy Neighbor | June 30th, 2016 | Github Repo | Deployed Here

Description

Responsive web application for checking new construction and development in areas and neighborhoods within Seattle

Project Employs:
  • Single page architecture
  • Google Maps API with Places library
    • and the data.seattle.gov api to pull the data.
  • Node.js with Express for the server
  • WebDB for the database (SQLite)
  • City of Seattle API for gathering construction permit data
  • MVC Architecture
  • Deployed on Firebase
